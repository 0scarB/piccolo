Authentication
==============

Piccolo ships with some authentication support out of the box.

-------------------------------------------------------------------------------

Registering the app
-------------------

Make sure ``'piccolo.apps.user.piccolo_app'`` is in your ``AppRegistry`` (see
:ref:`PiccoloProjects`).
